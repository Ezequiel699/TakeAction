# TakeAction
TakeAction es un simulador de vida minimalista creado en C#.
El usuario toma decisiones cotidianas como hacer ejercicio, procrastinar, estudiar o meditar, y cada acción afecta las estadísticas del personaje según un resultado aleatorio.

El objetivo es mostrar cómo pequeñas decisiones pueden influir en tu energía, motivación, enfoque y nivel de estrés.

Cómo usarlo:

1. Ejecuta el programa.
2. Elige una opción del 1 al 5.
3. Mira cómo cambian tus estadísticas.
4. Sigue tomando decisiones.
